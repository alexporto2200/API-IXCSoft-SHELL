# API-IXCSoft-SHELL
API shell script

Realize o download ou clone o repositório.

## Dependências 

apt install curl jq git
  
## Instalação

git clone https://github.com/alexporto2200/API-IXCSoft-SHELL.git;

cd API-IXCSoft-SHELL/;

chmod +x api.sh;

## Testando e verificando a ajuda

./api.sh --help;
